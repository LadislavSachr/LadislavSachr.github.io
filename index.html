    Aplikace E-commerce a To-Do list jsou nasazeny na platformě Render (free verze) a proto může dojít až k 1 minutovém spoždění startu.

    E-commerce

    E-shop zaměřený na prodej her a herního příslušenství. Uživatel se může zaregistrovat, přihlásit, přidávat položky do košíku, dokončit objednávku a provést platbu prostřednictvím Stripe Checkout. Po přihlášení má uživatel možnost zobrazit historii svých objednávek.

    To-Do list

    Aplikace se zaměřením na produktivitu, která umožňuje uživateli přidávat a mazat úkoly. Uživatel má možnost je také filtrovat podle ruzných kategorií.

    React, Node.js (Express.js), PostgreSQL

    Netflix hodnocení

    Google Chrome rozšíření určené pro Netflix. Využívá OMDb API k získávání hodnocení od IMDb a Rotten Tomatoes pro konkrétní seriál nebo film. Následně upravuje DOM a zobrazuje tato hodnocení přímo pod názvem vybraného titulu.

    JavaScript, HTML, CSS

    Generátor hesel

    Google Chrome rozšíření pro generaci nahodného hesla. Uživatel má plnou kontrolu nad tím, jaké znaky a symboly má heslo obsahovat, stejně jako nad jeho délkou. Během generování se síla hesla automaticky vyhodnocuje a uživateli ukazuje.
